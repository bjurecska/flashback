# flashback
Javascript based memory game

Game Dependencies:

main.html: Main game site. Provides the basic outlay of the game in html.

style.css: main.html's style sheet. Also includes the animation for the game.

scripts.js: main.html's javascript source. All important gameplay procedures and variables can be found here.
Important functions included in scripts.js:
-Flip: Responsible for main gameplay. What happens every time a user flips a card
-preLoadListeners: click listeners to continually update the moves and star display
-game(object): includes various properties and functions crucial to the game
-newGame: Initializes and resets the game.
